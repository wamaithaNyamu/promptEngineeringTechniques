<!-- GETTING STARTED -->
### Prerequisites

1. A Gemini API Key
2. NodeJS 20+

### Getting started
Install dependencies with npm
```sh
npm install
```
Create a `.env` file
```sh
touch .env
```
Add the Gemini api key in the .env
```sh
GEMINI_API_KEY=<add-key-here>
```
Run the code
```sh
npm run start
```
